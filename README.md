# 🇫🇮 Finland Tourism Website

[![Deploy to Render](https://render.com/images/deploy-to-render-button.svg)](https://finland-tourism.onrender.com/)

A modern tourism website showcasing the beauty of Finland. It provides detailed information about famous destinations, local culture, and travel guides for exploring the land of Santa Claus.

**🔗 Live Demo:** [https://finland-tourism.onrender.com/](https://finland-tourism.onrender.com/)

## ✨ Key Features
- 📍 **Discover Destinations:** Detailed guides on Helsinki, Lapland, Rovaniemi, and more.
- 📸 **Stunning UI/UX:** High-quality imagery with smooth scrolling and animations.
- 📱 **Responsive Design:** Fully optimized for Mobile, Tablet, and Desktop screens.
- 🗺️ **Interactive Maps:** Integrated maps to help users easily find locations.
- 📧 **Contact/Newsletter:** A functional form for user inquiries and newsletter subscriptions.

## 🛠 Tech Stack
- **Frontend:** HTML5, CSS3, JavaScript (React/Vue/Vanilla).
- **Deployment:** GitHub & Render.

## 🚀 Local Setup

To run this project locally on your machine, follow these steps:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/yourusername/finland-tourism.git
   cd finland-tourism
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the development server:**
   ```bash
   npm start
   ```
   Open [http://localhost:3000](http://localhost:3000) in your browser to view the app.

## 📦 How to Deploy on Render from GitHub

To set up continuous deployment so your website updates automatically whenever you push to GitHub:

1. **Push your code to GitHub:**
   - Create a new repository on GitHub.
   - Push your project files to the `main` branch.

2. **Configure Render:**
   - Log in to your [Render.com](https://render.com/) account.
   - Click on **New +** -> **Static Site** (for static HTML/React/Vue) or **Web Service** (if using a Node.js backend).
   - Connect your GitHub account and select the `finland-tourism` repository.

3. **Deployment Settings:**
   - **Build Command:** `npm run build` (leave blank if it's plain HTML/CSS).
   - **Publish Directory:** `dist`, `build`, or `.` (depending on your project structure).

4. **Deploy:**
   - Click **Create Static Site**. Render will automatically build and deploy your site, providing you with an `*.onrender.com` URL.
